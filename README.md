## 📌 Descripción / Description
**ES:** Este dashboard interactivo analiza el potencial económico de Venezuela usando datos públicos sobre recursos naturales, producción histórica, comparaciones internacionales y escenarios de inversión. Permite comprender de forma clara el impacto de distintas estrategias de inversión y visualizar oportunidades clave para el crecimiento económico.  

**EN:** This interactive dashboard analyzes Venezuela's economic potential using public data on natural resources, historical production, international comparisons, and investment scenarios. It allows users to clearly understand the impact of different investment strategies and visualize key growth opportunities.

## 🎯 Objetivos del proyecto / Project Goals
**ES:**  
- Mostrar el potencial económico de Venezuela de manera interactiva  
- Comparar recursos y sectores con otros países  
- Simular escenarios de inversión y su impacto en PIB, empleo y exportaciones  
- Presentar datos complejos de forma simple y clara  

**EN:**  
- Show Venezuela's economic potential interactively  
- Compare resources and sectors with other countries

## 📊 Contenido del dashboard / Dashboard Content
**ES:**  
1. Recursos naturales de Venezuela con gráficos interactivos y explicaciones sencillas  
2. Producción histórica de petróleo, minería y PIB  
3. Comparación internacional con España, Noruega, Chile, México y Perú  
4. Simulador de escenarios de inversión (turismo, petróleo, minería)  
5. Mapa interactivo de Venezuela con regiones de recursos destacadas  
6. Sección de insights y políticas económicas con explicaciones en lenguaje simple  

**EN:**  
1. Venezuela’s natural resources with interactive charts and beginner-friendly explanations  
2. Historical production of oil, mining, and GDP trends  
3. International comparison with Spain, Norway, Chile, Mexico, and Peru  
4. Investment scenario simulator (tourism, oil, mining)  
5. Interactive map of Venezuela highlighting key resource regions  
6. Policy insight section with plain-language explanations
